comandos flex

COMANDOS DE FLEX EN CSS

  1. DISPLAY: define un contenedor flexible, permitiendo alinear a sus hijos de la forma que deseemos

  2. Flex-direction: direccion de los items en columna o fila

  3. Flex-wrap: Determina si los items se ajustan en linea o no

  4. flex-flow:Flex-direction+Flex-wrap

  5. justify-content:alineacion horizontal de los items

  6. align-items: alineacion vertical de los items

  7. align-content: alinea los items de forma vertical y horizontal

  8. flex-grow: define en que proporcion crece un item

  9. flex-basis: define el tamano de un item antes de distribuir el espacio restante

  10. flex-shrink: Define la capacidad de un objeto para contraerse si es necesario

  11. order: Permite ordenar items en una forma diferente a como esta establecido

  12. align-self: Permite alinear de forma independiente a un item

comandos grid

COMANDOS GRID EN CSS

  1. DISPLAY: Determina que el contenedor va a ser un grid

  2. GRID-TEMPLATE: Define el ancho y alto de columnas y filas del grid

  3. GRID-GAP: Define el tamano del espacio entre las columnas y filas del grid

  4. JUSTIFY-ITEMS: Justifica las columnas o filas del grid

  5. ALIGN-ITEMS: Alinea las columnas o filas del grid

  6. JUSTIFY-CONTENT:Justifica las columnas o filas del grid

  7. ALIGN-CONTENT:Alinea las columnas o filas del grid

  8. GRID-AUTO-FLOW: Coloca automaticamente elementos que no estan explicitamente ubicados

  9. GRID-ROW/GRID-COLUMN/GRID-COLUMN+GRID-ROW: Indica la ubicacion basada en filas o columnas dentro del grid

  10. JUSTIFY-SELF/ALIGN-SELF: Alinea vertical u horizontalmente el contenido de una cuadricula
